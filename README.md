## Installation

Clone the project form github into your local machine

```bash
  git clone https://github.com/shrikant9024/thence-task.git
```

cd into the project

```bash
  cd thence
```

Install the dependencies

```bash
  npm i
```

Run the project locally

```bash
 npm run dev
```
